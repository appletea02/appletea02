정리

* 웹 브라우저가 서비스를 요청request 하면 웹 서버가 서비스를 보내주는response 역할.
여기서 웹 브라우저는 요청하는 client고 웹 서버는 서비스를 하는 server.
client가 게임 또는 그것이 설치된 폰이라면 server는 게임회사.
* hosts는 인터넷이 설치된 하나하나의 컴퓨터들.
웹호스팅은 웹서버를 운영하기 위한 호스트를 빌려주는 서비스.
* http://127.0.0.1/index.html 은 내 컴퓨터 ip.
여기서 127은 Internet Protocol ADRESS, 즉 ip 주소를 말함.
공유기를 사용한다면 네트워크 및 공유센터 - 연결 - 자세히 - IPc4 주소.index.html
* http 는 hyper text(웹페이지) transfer(전송) protocol(약속) 의 약자.
